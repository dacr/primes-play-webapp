@(message: String, valuesCount:Int, primesCount:Int, highestPrime:Long) @main("Welcome to Play") {

@message

The database cache contains @valuesCount already checked values, with @primesCount primes found. The highest found prime is @highestPrime

The API

  • check/$num : to test if $num is a prime number or not
  • slowcheck/$num/$secs : to test if $num is a prime number or not, and wait $secs seconds at server side, this is for test purposes
  • prime/$nth : to get the nth prime, 1 -> 2, 2->3, 3->5, 4->7
  • factors/$num : to get the primer factors of $num
  • primes/$to : list primes up to $to
  • primes/$form/$to : list primes from $from to $to
  • populate/$upTo : populate the database up to the specified value. Take care it calls a synchronized method.
  • ulam/$size : Dynamically draw an ulam spiral with the give $size. Take care of your CPUs and Heap ; this is a server side computation
}
